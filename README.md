# user-defined-funtions-problems
For the last 2 days, I’ve been learning Python functions and today I practiced them by building 8 small programs based on functions and recursion.

Honestly, today felt like one more real step forward because functions started making coding feel more organized and reusable instead of writing everything again and again.

Programs I built today:
• Greatest of Three Numbers using Functions
• Celsius to Fahrenheit Converter
• Preventing print() from adding a new line
• Recursive Sum of Natural Numbers
• Recursive Star Pattern
• Inches to Centimeters Converter
• Remove Word from List & Strip it
• Multiplication Table using Function

While solving these, I practiced:
*functions
*recursion
*return statements
*parameters & arguments
*loops inside functions
*pattern logic
*problem solving

Recursion especially felt confusing at first, but after tracing how the function keeps calling itself, it actually became interesting. Feels good seeing logic slowly start making sense instead of just staring at code like ancient symbols.

Still learning step by step and trying to stay consistent by building programs daily.
